The app acts as a sort of keep-up tool with the trackmania community.

The two main current functionallities are most popular videos and most popular tracks.

Youtube API is used to get the most trending videos about the game for the last week.
TMX API is used to get the most trending tracks amongst the community for the current week.

The app has fully functional register and login functioanllities with correct username and password checking behaviour.
